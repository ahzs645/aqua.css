Wave Scrollbar Demo

This scrollbar uses the same liquid wave effect as the progress bar, creating a cohesive Aqua experience.

The wave pattern in the thumb creates a sense of depth and movement as you scroll through the content.

Notice how the wave stays fixed relative to the track, revealing different parts of the pattern as the thumb moves.

You can drag the thumb directly or scroll the content area to see the effect.

The design matches the classic Mac OS X Aqua interface with its glossy, three-dimensional appearance.

Try switching between "Arrows Together" and "Arrows Split" modes!

This implementation responds to theme changes.
